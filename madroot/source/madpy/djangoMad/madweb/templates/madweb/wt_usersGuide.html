Madrigal user's guide (How do I access Madrigal data?)

There are two basic ways to access Madrigal data - through a web browser, or through the API using the programming languages Matlab, IDL, Python, or Fortran95. The Madrigal web interface will also allow you to create the most common commands you would need to run these API's. That is, with just a few click you can create a command to download whatever files you want from any number of experiments in any format. You can also create a command to download just the parameters you want with any filter you might want.

This user's guide consists of a brief tutorial on using the web interface, and a tutorial and a reference manual for remote data access. The last section also documents some command line interfaces into Madrigal.
