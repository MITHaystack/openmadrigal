Creating direct links to experiments/files

There are times you as administrator might want to create urls that directly take a user to a particular experiment, or to a particular file within an experiment. Below HOME refers to the url to your site's home page. In all cases, if a user has not yet registered, they will first go to the register page, and then directly to the requested url.

To go directly to a particular Madrigal experiment, follow the example below:

HOME/showExperiment?experiment_list=experiments/2017/arg/27jul17

To go directly to a particular Madrigal experiment and activate the show plots/docs button, follow the example below:

HOME/showExperiment?experiment_list=experiments/2017/arg/27jul17&show_plots

To go directly to a particular Madrigal file, follow the example below:

HOME/showExperiment?experiment_list=experiments/2017/arg/27jul17&file_list=one_ion_aro20170727topg.000.hdf5

To go directly to a particular Madrigal file and activate the show plots/docs button, follow the example below:

HOME/showExperiment?experiment_list=experiments/2017/arg/27jul17&file_list=one_ion_aro20170727topg.000.hdf5&show_plots
